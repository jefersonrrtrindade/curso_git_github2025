Fluxo de trabalho Git local:


1 - git checkout -b nova branch

2 - cria ou atualiza arquivos

3 - git status

4 - git add "arq"

5 - git status

6 - git commit -m "minha mensagem"

7 - git checkout main

8 - git merge nova\_branch
