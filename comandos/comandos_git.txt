git init. - cria um reposit git na pasta atual
git status - mostra o status do trabalho de cada arquivo
git add nome_arquivo - adiciona arquiv para ser commit
git commit - m "msg" - faz o commit do que estava para ser salvo
git log -> histórico de todos os commits

## FLUXO DE TRABALHO 

1. criar o diretório do projeto
2. iniciar o git dentro do diretório criado (git init.)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3
